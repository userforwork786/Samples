Information related to Sample structure, json schema.

JSON Schema

- JSON Keys should not be changed. Otherwise, exception will be thrown.
- JSON values can be changed as required.
- 'Features' contains all the features based on groups i.e client-side, server-side.
- operations contains the list of operations/features in that group i.e client-side -> Expiration etc.
- 'Title' contains the name of the feature that is displayed on web end. 
- Feature's folder name should be same as the Title. Otherwise, exception will be thrown.
- Description is in the format that can be rendered as markdown in web end.
